https://github.com/zarazhangrui/codebase-to-course

## 功能特性

- 保留了核心功能
- 简化了代码结构
- 去除了复杂依赖
- 优化了代码质量

## 安装依赖

```bash
# Python项目
pip install -r requirements.txt

# Node.js项目
npm install
```

## 使用方法

```bash
# Python项目
python main.py

# Node.js项目
node index.js
```
